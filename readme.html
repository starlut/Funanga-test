Project Structure

The project is organized into frontend and backend components.

Frontend

  • frontend/index.html: Main entry point for the application.
  • frontend/login/login.html: Login page HTML.
  • frontend/login/style.css: Stylesheet for the login page.
  • frontend/login/script.js: JavaScript logic for the login page.
  • frontend/home/home.html: Welcome page HTML.
  • frontend/home/style.css: Stylesheet for the welcome page.
  • frontend/home/script.js: JavaScript logic for the welcome page.

Backend

  • backend/index.php: Main entry point for the backend, handles routing.
  • backend/src/config.php: Configuration file for database connection.
  • backend/src/controller.php: Handles different API routes.
  • backend/src/service.php: Contains the business logic, interacts with the database.

Frontend Implementation

The frontend is built using HTML, CSS, and JavaScript. The login page has a modern and user-friendly design. It makes an AJAX call to the backend API for user authentication.

Backend Implementation

The backend is implemented in PHP. It includes routing, a configuration file for database connection, and a service class for business logic. The controller.php file handles different API routes, and the service.php file interacts with the database for user authentication.

Database Structure

The database used is named funanga_test, and it contains a table named user_info with columns id, name, email, and password.
